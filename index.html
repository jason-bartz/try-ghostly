Features

What Ghostly does

Local Whisper

Transcription runs on-device with GPU acceleration on Apple Silicon. Audio doesn't go to a server.

Global shortcut

Hold to talk, release to paste. Works in Slack, VS Code, email, Figma — anywhere the cursor is.

Optional AI cleanup

Bring a key for OpenAI, Anthropic, Groq, or use Apple Intelligence. Fixes punctuation, formats lists, strips fillers.

Screenshot + ask

One shortcut grabs the screen, records your question, and pastes the vision model's answer.

Edit last paste

Press the edit shortcut, say "make it shorter" or "change Monday to Tuesday" — the previous output gets rewritten in place.

Per-app profiles

Swaps prompt and vocabulary based on the focused app. Terse for Slack, formal for email, code-aware for your editor.

Voice commands

Map spoken phrases to keystrokes. Say "approve" to send Enter, "reject" for Escape. Pairs well with Cursor and Claude Code.

"Scratch that"

Say it mid-sentence and everything before it drops, so you can self-correct without re-recording. The phrase is configurable.

Swappable engines

Whisper ships as default. Parakeet (Nvidia) and Moonshine are there if you want faster inference or English-only streaming.

How it works

Hold the shortcut, talk, release.

The transcript pastes into whatever field the cursor is in. Silences get skipped automatically.

01

Press your shortcut

Any keybind, anywhere. Push-to-talk or toggle. Voice activity detection skips the silences.

02

Talk

Whisper transcribes on-device. Self-correct with "scratch that." Say numbers out loud — AI cleanup handles conversion.

03

Release

The cleaned transcript pastes into the active app. No window switch, no copy step.

AI cleanup

Bring your own model.

Connect an API key for OpenAI, Anthropic, Groq, OpenRouter, Cerebras, Z.AI, or any OpenAI-compatible endpoint. Ghostly runs the transcript through a model before pasting — fixing punctuation, formatting lists, converting "twenty-five percent" to 25%, stripping fillers.

On Apple Silicon, pick Apple Intelligence for on-device cleanup with no key and no network.

Without a key, nothing breaks — you get the raw Whisper output.

Install Ghostly

Grab the latest .dmg from releases, drag Ghostly to Applications, launch. It'll ask for microphone and accessibility access. Pick a shortcut in Settings. First run downloads a transcription model (500 MB–1.6 GB depending on which one you pick). After that, dictation works offline.

FAQ

Questions

Does my audio leave my Mac?▾

No. Transcription runs on-device via Whisper. If you enable optional AI cleanup with a cloud provider, only the resulting text is sent — never audio. Apple Intelligence keeps cleanup on-device too.

Is it free?▾

Yes. Ghostly is free to download and use.

What are the system requirements?▾

macOS 10.15 Catalina or later. Apple Silicon recommended for GPU-accelerated inference; Intel works. Microphone and accessibility permissions required.

Does it work offline?▾

Yes. After the first-run model download, dictation works offline. Optional AI cleanup needs a network connection unless you pick Apple Intelligence.

Windows or Linux?▾

Ghostly targets macOS only.
